# Scripts

## Folder structure

├── pep_slicing/

├──pep_mimicking/

├──scripts_md/

├──README.md

- **pep_slicing/** – folder contains two python script file to generate peptides (9–12 mers) with or without anchor rules.
- **pep_mimicking/** – folder contains two python script file toIdentify HLA-binding peptides and mimicry candidates via alignment.
- **scripts_md/** – folder contains two python script file related to md run from optimization (Amber) to analysis (GROMACS).
