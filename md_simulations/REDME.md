
# Molecular Dynamics Simulations
Contains all input, setup, and post-processing data for 1 µs MD simulations using Amber and GROMACS.

- **tleap_input/** – AMBER parameter and coordinate files generated using tleap.
- **amber_to_gromacs/** – Converted topology and coordinate files (via ParmEd).
- **gromacs_inputs/** – MDP parameter files for minimization, NVT, NPT, and production runs.
**index_files/** – all index files after md run
- **Movie/** – MD visualization videos of each system.

- **scripts/** – Bash scripts for MD execution and trajectory correction.
